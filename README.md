# myapp-boilerplate

Create a boilerplate web app template, the main folder is named MyApp, inside this folder, there are folders frontend and backend. Use react with typescript and tailwind for the frontend and Django python for the backend. Create a simple Homepage listing some products that are stored on the backend.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/myapp-boilerplate.git
cd myapp-boilerplate
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
